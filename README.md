# VØID AVIATOR RESEARCH PRO

Plateforme open-source de recherche algorithmique et de simulation de stratégies pour les jeux de crash (type Aviator).

## Fonctionnalités
- **Analyse Multi-Stratégies** : Martingale, Fibonacci, D'Alembert, Trend Following.
- **Moteur de Simulation** : Basé sur les principes de Provably Fair.
- **OCR Analysis Interface** : Interface prête pour l'intégration de reconnaissance optique de caractères.
- **Mobile First** : Conçu pour être utilisé sur smartphone.

## Disclaimer
Ce projet est à but **éducatif et de recherche uniquement**. L'utilisation de ces algorithmes sur des plateformes de jeux réelles comporte des risques financiers. Nous ne garantissons aucun résultat positif.

## Installation (Hébergement GitHub Pages)
1. Créez un repository sur GitHub.
2. Déposez `index.html`.
3. Activez GitHub Pages dans les réglages.
